# Research: Colorado Wildfire Code AI Compliance Gap
## Catherine "Code" Chen | Policy & Regulation Beat

### Thesis
Colorado's Wildfire Resiliency Code (CWRC) takes effect July 1, 2026 — the second statewide mandatory WUI code after California. Local jurisdictions were supposed to adopt it by April 1, 2026. The Colorado Municipal League flagged "inaccurate maps, lack of data tools, and staff capacity issues." The code itself is solid. The enforcement infrastructure doesn't exist yet. AI tools could close the gap — but nobody's deploying them at the jurisdictional level where they're needed most.

### Kill Test
Does this help someone building or buying a home? YES. If you're building in Colorado's WUI zone (1.1M+ homes), you need to know what the code requires, what it costs, and whether your jurisdiction can actually enforce it. If you're buying a home that was "code-compliant," you need to know whether that review was thorough or rubber-stamped.

### Primary Sources

**1. Colorado Wildfire Resiliency Code (CWRC)**
- Adopted statewide July 1, 2025 by the Wildfire Resiliency Code Board (SB 23-180)
- Local jurisdictions must adopt by April 1, 2026; enforcement begins July 1, 2026
- Two risk classes: Class 1 (higher risk) and Class 2 (moderate risk)
- Requirements: fire-resistant roofing (Class A), noncombustible exterior walls, ember-resistant vents (ASTM E2886), defensible space zones
- Construction cost increase: ~2.7% (CPR/Colorado reporting)
- Source: Mesa County adoption (mesacounty.us), Durango (durangoco.gov), Lyons (townoflyons.com)

**2. Colorado Municipal League (CML) Implementation Concerns**
- "Inaccurate maps, lack of data tools, and staff capacity issues" flagged as barriers
- Many smaller jurisdictions lack plan review staff trained in WUI compliance
- Wildfire risk mapping is "controversial" — affects insurance and property values
- Source: cml.org wildfire resiliency code update

**3. Headwaters Economics — State WUI Code Tracker**
- As of April 2026: only California and Colorado have mandatory statewide WUI codes
- Utah passed law requiring adoption by 2029
- Oregon and Washington attempting but not yet in effect
- Key insight: "homes and structures act as sources of ignition, spreading fire from property to property" — individual action insufficient, need neighborhood-wide adoption
- Research: homes built to WUI code are "much more likely to survive" — Nature study (2025)
- Source: headwaterseconomics.org

**4. Camp Fire Data (SGH Engineering)**
- Post-2018 Camp Fire analysis: homes built after 2008 under California's Chapter 7A codes were **3x more likely to survive**
- 85,000 residents displaced, 18,804 structures destroyed
- Code compliance = survival. Non-compliance = total loss.
- Source: sgh.com wildfire construction practices

**5. Washington State Cost Data**
- SB 6279 debate: builders warn of **$24,000 per-home cost** from premature WUI code adoption
- Colorado's 2.7% figure on a $500K home = ~$13,500
- Gap between WA estimate and CO estimate suggests wide variance in interpretation and local enforcement
- Source: citizenportal.ai (WA committee debate)

**6. ICC International WUI Code**
- Model code adopted/adapted by jurisdictions
- Covers: ignition-resistant construction, defensible space, fire suppression, road standards, water access
- Over 46 million US homes in WUI zones (ICC data)
- 100,000+ structures destroyed by wildfire since 2005
- Source: iccsafe.org

**7. AI Tools in This Space**
- **WIMPLE** (Charles River Analytics): Hybrid AI wildfire risk assessment for homeowners/communities. Uses NASA/NOAA/USFS data. Risk simulations + mitigation recommendations.
- **ArcGIS/Esri**: Field Maps + Experience Builder for community wildfire protection plans. Not construction-specific.
- **UMD Fire Protection Engineering**: AI wildfire forecasting from aerial imagery. Research stage.
- **Generic AI plan review tools** (PermitFlow, Symbium, Caltron): Could theoretically check WUI compliance but none specifically trained on CWRC provisions.
- **Gap**: No jurisdictional-level AI tool exists that checks construction plans against WUI code. Plan reviewers are doing it manually — if they're doing it at all.

### Novel Contribution (Calculation)
Cost-benefit for a Colorado WUI homebuyer:
- Average new home price in Colorado WUI zone: ~$550K (estimate from Zillow/Redfin data)
- WUI compliance cost adder: 2.7% = ~$14,850
- Insurance savings: Colorado homeowners in fire-prone areas paying $3,000-$8,000/year premiums. WUI-compliant homes can qualify for 10-30% discounts = $300-$2,400/year
- Payback period on compliance investment: 6-50 years on insurance alone
- But: 3x survival rate means avoided total loss ($550K) * probability of fire = massive expected value gain
- Colorado's WUI fire probability: Marshall Fire (2021) hit 1,084 homes in WUI. ~6,200 homes in Boulder County WUI zone. 17% destruction rate in one event.
- Expected value of code compliance per year (simplified): 0.5% annual fire probability * 67% loss avoidance rate * $550K home value = ~$1,842/year. Compliance cost amortized over 30-year mortgage: ~$495/year. **Net benefit: ~$1,347/year.**

### Counterargument (at full strength)
The 2.7% cost figure is based on Colorado's own estimates — not independently audited. Washington builders say $24,000, which on a $550K home is 4.4%. If compliance costs are higher than advertised, the cost-benefit math gets shakier, especially in lower-risk Class 2 zones. And CML's own admission that maps are "inaccurate" means some homeowners will be paying compliance costs for risks that don't match their actual parcel-level exposure. The code also doesn't apply to existing homes — the 1.1M already-built homes in Colorado's WUI get no safer. A new code that protects only new construction while leaving existing housing stock untouched is fighting the last war.

### Limitations
- 2.7% cost estimate is state-provided, not independently verified
- Insurance premium data varies wildly by carrier; "10-30% discount" is industry range, not guaranteed
- Fire probability calculation is rough — uses Marshall Fire as anchor but that was an extreme event
- AI plan review tools in this space are nascent; none have been tested at scale for WUI compliance
- No data on how many Colorado jurisdictions have actually adopted by the April 2026 deadline
